# 공지 · 정책

아래 3종 공지는 전 채널에 일관 적용되며, 사이트 푸터에도 요약 노출됩니다.

## 1 · 심사 런타임 인터넷 차단

경연 **런타임은 외부 네트워크에 접근할 수 없습니다.** 심사 중 공개 API 호출은
**허용되지 않습니다.**

> **빌드 시점은 인터넷 사용 가능합니다.** 모델 가중치·의존성은 빌드 시점에 이미지에
> baking 하세요(이때 인터넷 가능). 런타임에는 **다운로드·외부 API 호출 금지**.

## 2 · 제3자 자산 라이선스 · Attribution

MARC 는 **오픈소스 소프트웨어(OSS)** 및 **USD** 콘텐츠 등 제3자 자산을 사용합니다.
해당 라이선스와 출처를 고지하며, 이는 USD 라이선스 검토 결과를 반영합니다.

- OSS / USD 라이선스·attribution 통합 목록은 스타터킷·개발자 가이드에 제공됩니다.
- 참가자는 번들 및 다운로드한 모든 자산의 라이선스를 준수해야 합니다.

## 3 · 대회 배경 장소(playground)는 변경될 수 있음

공개되는 배경 USD 는 **연습용 `chungmu`** 환경 1종뿐입니다. **실제 대회는 다른
배경에서 진행될 수 있습니다** — 무대 과적합/사전 노출 방지를 위함입니다.
