# 공지

## 1 · 심사 런타임 인터넷 차단

참가자가 개발한 프로그램은 **상용 API(클라우드 LLM 등 외부 AI API)를 사용할 수 없습니다.**
채점은 **외부 네트워크가 차단된 런타임**에서 이뤄지므로, 실행 중에는 어떤 외부 API 호출이나
인터넷 접속도 허용되지 않습니다.

> **빌드 시점은 인터넷 사용 가능합니다.** 모델 가중치·의존성은 빌드 시점에 이미지에
> baking 하세요(이때 인터넷 가능). 런타임에는 **다운로드·외부 API 호출 금지**.

## 3 · 제3자 자산 라이선스 공지

MARC 는 **오픈소스 소프트웨어(OSS)** 및 **USD** 콘텐츠 등 제3자 자산을 사용합니다. 해당
라이선스와 출처를 고지합니다.

- **NVIDIA Isaac Sim** — 시뮬레이션 플랫폼. 라이선스상 프리빌트 이미지는 재배포하지 않으며, 참가자가 본인 명의로 base 이미지를 받아 로컬에서 빌드합니다. 이때 base 이미지를 내려받으려면 **NVIDIA 계정(NGC)** 이 필요합니다.
- **ROS 2 (Humble)** — 로봇 제어·센서 통신용 표준 인터페이스.
- **참가자 SDK (`marc-sdk`)** — MIT 라이선스.
- **배경·캐릭터 USD** — 연습용 chungmu 배경 및 공개 자산(캐릭터는 MakeHuman(CC0) 등으로 대체).

전체 라이선스·attribution 통합 목록은 스타터킷의 `NOTICES.md` 및 개발자 가이드에 제공됩니다.
참가자는 번들 및 다운로드한 모든 자산의 라이선스를 준수해야 합니다.

## 4 · 대회 배경 장소(playground)는 변경될 수 있음

참가자에게 공개되는 대회 배경(playground)은 연습용 **`chungmu`** 환경 한 가지뿐입니다.
**실제 대회는 이와 다른 배경에서 진행될 수 있습니다.** 이는 특정 배경에만 맞춰 과도하게
최적화(과적합)하는 것을 막고, 실제 무대가 미리 노출되는 것을 방지하기 위한 조치입니다.
따라서 여러분의 에이전트는 **처음 보는 환경에서도 잘 동작하도록** 일반화하여 개발하는 것이
좋습니다.
